Daily-Todos ინსტრუქცია
---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

1. ნაბიჯი - array-ების დამატება
   - გამოვიყენოთ useState([]) რათა შევქმნათ todos ჩამონათვალი (მასში შემევალი ინფორმაცია იქნება ცვლადი ამიტო გვჭირდება state variable). (მაგალითად: const [todos, setTodos] = useState([]);)
   - შევქმნათ მეორე array რომელშიც წინასწარ იქნება განსაზღვრული დატა შესაბამისად გამოვიყენოთ ჩვეულებრივი array. (სახელი მაგალითად array-ს presetTodos)

2. ნაბიჯი - ფუნქციების დამატება
   - დავამატოთ ფუნქცია რომელიც ღილაკის ხელის დაჭერაზე დაამატებს ახალ item-ს todos array-ში.
   - გამოვიყენოთ spread operator (...) რომელიც დააბრუნებს ახლანდელ array-ს და დაამატებს ახალ item-ს. (! გვჭირდება იმისათვის რომ არ დავკარგოთ მოცემული array-ის ყველა item-ი)
   - ყველა ახალი item-ისთვის გვჭირდება ახალი ობიექტის შექმნა რომელსაც შემდეგი ველები ჰქონდეს: id: nanoid(), text: todo. (ანუ todos array იქნება ობიექტქბის სიმრავლე {id:, text:} )
   - id-ს თვის გამოვიყენოთ nanoid() ფუნქცია რომელიც გვიბრუნებს უნიკალური მნიშვნელობას.
   - შევქმნათ წაშლის ფუნქცია რომელიც იყენებს filter() და გვიბრუნებს array რომელიც არ შეიცავს ჩვენ მიერ წაშლილ item-ს.

3. ნაბიჯი - todos ჩამონათვალის ჩვენება
  - todos item-ები დავარენდეროთ .map() ფუნქციის დახმარებით. (გასატესტად შეგიძლიათ დროებით შეავსოთ todos array-ის ამ ობიქეტებით : {id: 1, text: "დატა 1"}, {id: 2, text: "დატა 2"}, {id: 3, text: "დატა 3"})
  - conditional rendering - ის დახმარებით ჩამონათვალის ჩვენება ან ცარიელი შეტყობინების ჩვენება. (ამის გატესტვაც შეგიძლიათ უბრალოც ცარიელი todos array-ის გამოყენებით)
  - არ დაგვავიწყდეს key prop-ში უნიკალური id გადაცემა. (გამოვიყენოთ todo.id)

4. ნაბიჯი - წინასწარ განსაღზრული ჩამონათვალის ჩვენება
  - .map() ფუნქციის დახმარებით ჩამონათვალის ჩვენება.

5. ნაბიჯი - EVENT HANDLERS დამატება
   - გამოვიყენოთ onClick={() => functionName()} და სწორდ ადგილას გამოვიძახოთ ჩვენ მიერ შექმნილი ფუნქციები.

REMINDERS:
----------
- შევინახოთ ყველა todo ობიექტის სახით: { id: nanoid(), text: todo }
- გამოვიყენოთ nanoid() ფუნქცია რომელიც გვიბრუნებს უნიკალური მნიშვნელობას.
- key prop-ში უნიკალური id-ის გადაცემა. (გამოვიყენოთ todo.id)
- [...todos, { id: nanoid(), text: todo }] - გამოვიყენოთ spread operator რომელიც დააბრუნებს ახლანდელ array-ს და დაამატებს ახალ item-ს.
- თუ საჭირო გახდა გადაიმეორეთ ინტერნეტში filter() და map() ფუნქციების გამოყენება.
